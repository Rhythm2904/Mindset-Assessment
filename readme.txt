Mindset Assessment: Are You Disillusioned?

This project is a web-based assessment tool designed to evaluate different mindsets or attitudes, focusing on idealism, disillusionment, cynicism, and hopefulness. It allows users to answer a series of questions and tracks their progress through visual progress bars.

Features

- Category Progress Bars: Visual representation of progress across different mindset categories.
- Interactive Question Slider: Allows users to slide and select their responses ranging from "Strongly Disagree" to "Strongly Agree".
- Navigation Buttons: Buttons to navigate between questions ("Prev" and "Next").
- Responsive Design: Optimized for various screen sizes and devices.
